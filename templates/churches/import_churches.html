CSV Format Instructions

Your CSV file should include the following columns:

Required Fields:
  • name - Church name
  • city - City
  • state - State/Province
Optional Fields:
  • denomination - Religious denomination
  • website - Church website URL
  • email - Contact email address
  • phone - Contact phone number
  • address_line1 - Street address line 1
  • address_line2 - Street address line 2
  • postal_code - ZIP/Postal code
  • country - Country (defaults to "United States")
  • size - Church size (small, medium, large, or mega)
  • pipeline_stage - Pipeline stage (new, contacted, meeting_scheduled, proposal, committed, partnership, inactive)
  • priority - Priority level (low, medium, high)
  • notes - Additional notes
Example CSV:
name,denomination,city,state,email,phone,pipeline_stage,priority

"St. Mary's Catholic Church","Catholic","Peoria","IL","parish@stmarys.org","555-456-7890","meeting_scheduled","medium"
